# Space Docking System using Arduino

This project demonstrates a prototype of a space docking system using Arduino, sensors, and a servo motor.

## Components Used
- Arduino Uno
- Ultrasonic Sensor
- IR Sensor
- Servo Motor
- Buzzer
- Breadboard
- Battery

## Working
The sensor detects the distance between the spacecraft and docking station. 
When the distance becomes safe, the servo motor activates and completes the docking process.
